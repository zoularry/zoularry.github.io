# FAQ

## 🔐 Compte & Sécurité
- **Q : Que dois-je faire si j'oublie mon mot de passe ?**
  > Vous pouvez réinitialiser votre mot de passe sur l'écran de connexion. Cliquez sur l'icône du cadenas, et le système vous demandera de vous authentifier en tant que propriétaire de l'appareil (via empreinte digitale ou reconnaissance faciale).
  > Si vous souhaitez désactiver complètement cette fonctionnalité, vous pouvez décocher "Autoriser la récupération du mot de passe" dans les paramètres.

- **Q : Que sont l'"Application Camouflage" et le "Mode Faux" ?**
  > Vous pouvez sélectionner "Application Camouflage" dans les paramètres pour changer l'icône de l'application en calculatrice ou autre outil courant, la rendant complètement "invisible" sur votre écran d'accueil.
  > Avec le "Mode Faux" activé, vous pouvez configurer un geste/code PIN factice. Lorsque vous vous connectez avec ces identifiants factices, l'application entre dans un espace factice complètement différent, vide ou non sensible, utilisé pour gérer les situations où d'autres vous forcent à montrer l'application.
  
- **Q : Où sont stockés mes fichiers privés ?**
  > Tous les fichiers cachés sont stockés en toute sécurité dans le dossier `.pg` de la mémoire interne de votre appareil. Ne supprimez pas ce dossier manuellement, car cela entraînerait une perte de données.

## 📸 Gestion des Médias
- **Q : Comment cacher rapidement des photos ou des vidéos ?**
  > Vous pouvez sélectionner des fichiers dans votre galerie système, appuyer sur "Partager", puis sélectionner "Hide Something" pour les cacher en quelques secondes.

- **Q : Quels formats sont supportés ?**
  > Nous supportons presque tous les formats d'image et de vidéo courants, y compris la vidéo 4K Ultra HD.

- **Q : Que sont les Auto-Clips et la Lecture Aléatoire ?**
  > **Auto-Clips** capture automatiquement les moments forts des vidéos longues pour un affichage aléatoire ; la **Lecture Aléatoire** offre une expérience de lecture au hasard, gardant vos moments de cinéma privé toujours nouveaux.

## 🌐 Navigateur & Multiplateforme
- **Q : Le navigateur privé est-il sûr ?**
  > Les données de notre navigateur intégré, y compris les favoris et l'historique de navigation, restent entièrement dans l'application et sont invisibles pour le monde extérieur, offrant une expérience en circuit fermé "rechercher-naviguer-télécharger-cacher".

- **Q : Comment accéder aux fichiers de mon téléphone depuis un ordinateur ?**
  > Après avoir activé l'"Accès PC/Mac", l'application lance un serveur HTTP temporaire sur le réseau local. Il suffit d'entrer l'adresse affichée dans le navigateur de votre ordinateur pour gérer les fichiers en toute sécurité, sans synchronisation cloud requise, zéro risque de fuite.
