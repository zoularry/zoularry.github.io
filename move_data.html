When setting up a new phone, most people use a data migration tool. How do you transfer the hidden files from HD SMTH?

If You Haven't Used the Phone's Migration Tool Yet

Standard transfer tools typically do not migrate hidden files. To ensure your hidden photos and videos are transferred to your new phone, follow these steps:

  • 1. Before using your phone's migration tool:
  • Unhide all files using HD SMTH on your old phone
  • 2. Then use your phone's migration tool
Reason: Migration tools only transfer visible photos and videos, not hidden data.

If You Already Used the Phone's Migration Tool

If you have already used the migration tool without restoring hidden files first, you will not find the hidden files on new phone.

There are two options to transfer them.

Simple Method (Phones Only)

  1. 1. On old phone: Open HD SMTH → Unhide all files
  2. 2. On both phones: Install and open Local Send App (Google Play Store)
  3. 3. On old phone: Select unhidden files in Local Send → Send
  4. 4. On new phone: Accept files in Local Send
  5. 5. After transfer: Open HD SMTH on new phone → Hide those files

Fast Method (Requires Windows PC)

  1. 1, Connect old phone to PC via USB cable
  2. 2. Select "File Transfer" mode on phone
  3. 3. On PC: Open File Explorer → This PC
  4. 4. Copy .pg folder from phone's root directory to PC
  5. 5. Disconnect old phone
  6. 6. Connect new phone to PC
  7. 7. Select "File Transfer" mode
  8. 8. Paste .pg folder to new phone's root directory
  9. 9. Disconnect new phone
  10. 10. Open HD SMTH on new phone

Al configurar un teléfono nuevo, la mayoría de las personas utiliza una herramienta de migración de datos. ¿Cómo transferir los archivos ocultos de HD SMTH?

Si aún no has usado la herramienta de migración del teléfono

Las herramientas estándar de transferencia normalmente no migran archivos ocultos. Para asegurarte de que tus fotos y videos ocultos se transfieran al nuevo teléfono, sigue estos pasos:

  • 1. Antes de usar la herramienta de migración de tu teléfono:
  • Desoculta todos los archivos usando HD SMTH en tu teléfono antiguo
  • 2. Luego usa la herramienta de migración de tu teléfono
Razón: Las herramientas de migración solo transfieren fotos y videos visibles, no datos ocultos.

Si ya usaste la herramienta de migración del teléfono

Si ya usaste la herramienta de migración sin restaurar primero los archivos ocultos, no encontrarás los archivos ocultos en el nuevo teléfono.

Hay dos opciones para transferirlos.

Método simple (solo para teléfonos)

  1. 1. En el teléfono antiguo: Abre HD SMTH → Desoculta todos los archivos
  2. 2. En ambos teléfonos: Instala y abre la aplicación Local Send (Google Play Store)
  3. 3. En el teléfono antiguo: Selecciona los archivos desocultos en Local Send → Enviar
  4. 4. En el nuevo teléfono: Acepta los archivos en Local Send
  5. 5. Después de la transferencia: Abre HD SMTH en el nuevo teléfono → Oculta esos archivos

Método rápido (requiere una PC con Windows)

  1. 1. Conecta el teléfono antiguo a la PC mediante un cable USB
  2. 2. Selecciona el modo "Transferencia de archivos" en el teléfono
  3. 3. En la PC: Abre el Explorador de archivos → Este equipo
  4. 4. Copia la carpeta .pg del directorio raíz del teléfono a la PC
  5. 5. Desconecta el teléfono antiguo
  6. 6. Conecta el nuevo teléfono a la PC
  7. 7. Selecciona el modo "Transferencia de archivos"
  8. 8. Pega la carpeta .pg en el directorio raíz del nuevo teléfono
  9. 9. Desconecta el nuevo teléfono
  10. 10. Abre HD SMTH en el nuevo teléfono

Ao configurar um telefone novo, a maioria das pessoas usa uma ferramenta de migração de dados. Como transferir os arquivos ocultos do HD SMTH?

Se você ainda não usou a ferramenta de migração do telefone

As ferramentas padrão de transferência geralmente não migram arquivos ocultos. Para garantir que suas fotos e vídeos ocultos sejam transferidos para o novo telefone, siga estas etapas:

  • 1. Antes de usar a ferramenta de migração do seu telefone:
  • Desoculte todos os arquivos usando HD SMTH no seu telefone antigo
  • 2. Em seguida, use a ferramenta de migração do seu telefone
Motivo: As ferramentas de migração transferem apenas fotos e vídeos visíveis, não dados ocultos.

Se você já usou a ferramenta de migração do telefone

Se você já usou a ferramenta de migração sem restaurar os arquivos ocultos primeiro, não encontrará os arquivos ocultos no novo telefone.

Há duas opções para transferi-los.

Método simples (apenas para telefones)

  1. 1. No telefone antigo: Abra HD SMTH → Desoculte todos os arquivos
  2. 2. Em ambos os telefones: Instale e abra o aplicativo Local Send (Google Play Store)
  3. 3. No telefone antigo: Selecione os arquivos desocultos no Local Send → Enviar
  4. 4. No novo telefone: Aceite os arquivos no Local Send
  5. 5. Após a transferência: Abra HD SMTH no novo telefone → Oculte esses arquivos

Método rápido (requer um PC com Windows)

  1. 1. Conecte o telefone antigo ao PC via cabo USB
  2. 2. Selecione o modo "Transferência de arquivos" no telefone
  3. 3. No PC: Abra o Explorador de Arquivos → Este Computador
  4. 4. Copie a pasta .pg do diretório raiz do telefone para o PC
  5. 5. Desconecte o telefone antigo
  6. 6. Conecte o novo telefone ao PC
  7. 7. Selecione o modo "Transferência de arquivos"
  8. 8. Cole a pasta .pg no diretório raiz do novo telefone
  9. 9. Desconecte o novo telefone
  10. 10. Abra HD SMTH no novo telefone

Lors de la configuration d'un nouveau téléphone, la plupart des gens utilisent un outil de migration de données. Comment transférer les fichiers cachés de HD SMTH ?

Si vous n'avez pas encore utilisé l'outil de migration du téléphone

Les outils de transfert standard ne migrent généralement pas les fichiers cachés. Pour vous assurer que vos photos et vidéos cachées sont transférées vers votre nouveau téléphone, suivez ces étapes :

  • 1. Avant d'utiliser l'outil de migration de votre téléphone :
  • Dévoilez tous les fichiers en utilisant HD SMTH sur votre ancien téléphone
  • 2. Ensuite, utilisez l'outil de migration de votre téléphone
Raison : Les outils de migration ne transfèrent que les photos et vidéos visibles, pas les données cachées.

Si vous avez déjà utilisé l'outil de migration du téléphone

Si vous avez déjà utilisé l'outil de migration sans restaurer d'abord les fichiers cachés, vous ne trouverez pas les fichiers cachés sur le nouveau téléphone.

Il existe deux options pour les transférer.

Méthode simple (pour téléphones uniquement)

  1. 1. Sur l'ancien téléphone : Ouvrez HD SMTH → Dévoilez tous les fichiers
  2. 2. Sur les deux téléphones : Installez et ouvrez l'application Local Send (Google Play Store)
  3. 3. Sur l'ancien téléphone : Sélectionnez les fichiers dévoilés dans Local Send → Envoyer
  4. 4. Sur le nouveau téléphone : Acceptez les fichiers dans Local Send
  5. 5. Après le transfert : Ouvrez HD SMTH sur le nouveau téléphone → Masquez ces fichiers

Méthode rapide (nécessite un PC Windows)

  1. 1. Connectez l'ancien téléphone au PC via un câble USB
  2. 2. Sélectionnez le mode "Transfert de fichiers" sur le téléphone
  3. 3. Sur le PC : Ouvrez l'Explorateur de fichiers → Ce PC
  4. 4. Copiez le dossier .pg du répertoire racine du téléphone vers le PC
  5. 5. Déconnectez l'ancien téléphone
  6. 6. Connectez le nouveau téléphone au PC
  7. 7. Sélectionnez le mode "Transfert de fichiers"
  8. 8. Collez le dossier .pg dans le répertoire racine du nouveau téléphone
  9. 9. Déconnectez le nouveau téléphone
  10. 10. Ouvrez HD SMTH sur le nouveau téléphone

عند إعداد هاتف جديد، يستخدم معظم الأشخاص أداة نقل البيانات. كيف تنقل الملفات المخفية من HD SMTH؟

إذا لم تكن قد استخدمت أداة نقل الهاتف بعد

عادةً ما لا تنقل أدوات النقل القياسية الملفات المخفية. لضمان نقل صورك ومقاطع الفيديو المخفية إلى هاتفك الجديد، اتبع هذه الخطوات:

  • 1. قبل استخدام أداة نقل الهاتف:
  • أظهر جميع الملفات باستخدام HD SMTH على هاتفك القديم
  • 2. ثم استخدم أداة نقل الهاتف
السبب: تنقل أدوات النقل فقط الصور ومقاطع الفيديو المرئية، وليس البيانات المخفية.

إذا كنت قد استخدمت أداة نقل الهاتف بالفعل

إذا كنت قد استخدمت أداة النقل دون استعادة الملفات المخفية أولاً، فلن تجد الملفات المخفية على الهاتف الجديد.

هناك خياران لنقلها.

طريقة بسيطة (للأجهزة فقط)

  1. 1. على الهاتف القديم: افتح HD SMTH → أظهر جميع الملفات
  2. 2. على كلا الهاتفين: قم بتثبيت وتشغيل تطبيق Local Send (متجر Google Play)
  3. 3. على الهاتف القديم: حدد الملفات غير المخفية في Local Send → إرسال
  4. 4. على الهاتف الجديد: قبول الملفات في Local Send
  5. 5. بعد النقل: افتح HD SMTH على الهاتف الجديد → أخفِ تلك الملفات

طريقة سريعة (تتطلب جهاز كمبيوتر يعمل بنظام Windows)

  1. 1. قم بتوصيل الهاتف القديم بالكمبيوتر عبر كابل USB
  2. 2. حدد وضع "نقل الملفات" على الهاتف
  3. 3. على الكمبيوتر: افتح File Explorer → This PC
  4. 4. انسخ مجلد .pg من الدليل الرئيسي للهاتف إلى الكمبيوتر
  5. 5. افصل الهاتف القديم
  6. 6. قم بتوصيل الهاتف الجديد بالكمبيوتر
  7. 7. حدد وضع "نقل الملفات"
  8. 8. الصق مجلد .pg في الدليل الرئيسي للهاتف الجديد
  9. 9. افصل الهاتف الجديد
  10. 10. افتح HD SMTH على الهاتف الجديد

Yeni bir telefon kurarken, çoğu kişi bir veri aktarma aracı kullanır. HD SMTH'den gizli dosyaları nasıl aktarırsınız?

Telefonun Aktarma Aracını Henüz Kullanmadıysanız

Standart aktarma araçları genellikle gizli dosyaları taşımaz. Gizli fotoğraf ve videolarınızın yeni telefona aktarılmasını sağlamak için şu adımları izleyin:

  • 1. Telefonunuzun aktarma aracını kullanmadan önce:
  • Eski telefonunuzda HD SMTH kullanarak tüm dosyaları görünür yapın
  • 2. Ardından telefonunuzun aktarma aracını kullanın
Sebep: Aktarma araçları yalnızca görünür fotoğraf ve videoları aktarır, gizli verileri değil.

Telefonun Aktarma Aracını Zaten Kullandıysanız

Eğer gizli dosyaları geri yüklemeden önce aktarma aracını kullandıysanız, yeni telefonda gizli dosyaları bulamazsınız.

Onları aktarmak için iki seçenek vardır.

Basit Yöntem (Yalnızca Telefonlar)

  1. 1. Eski telefonda: HD SMTH'yi açın → Tüm dosyaları görünür yapın
  2. 2. Her iki telefonda: Local Send uygulamasını yükleyin ve açın (Google Play Store)
  3. 3. Eski telefonda: Local Send'de görünür yapılan dosyaları seçin → Gönder
  4. 4. Yeni telefonda: Local Send'de dosyaları kabul edin
  5. 5. Aktarımdan sonra: Yeni telefonda HD SMTH'yi açın → Bu dosyaları gizleyin

Hızlı Yöntem (Windows PC Gerektirir)

  1. 1. Eski telefonu USB kablosuyla bilgisayara bağlayın
  2. 2. Telefonda "Dosya Aktarımı" modunu seçin
  3. 3. Bilgisayarda: Dosya Gezgini → Bu Bilgisayar'ı açın
  4. 4. Telefonun kök dizinindeki .pg klasörünü bilgisayara kopyalayın
  5. 5. Eski telefonun bağlantısını kesin
  6. 6. Yeni telefonu bilgisayara bağlayın
  7. 7. "Dosya Aktarımı" modunu seçin
  8. 8. .pg klasörünü yeni telefonun kök dizinine yapıştırın
  9. 9. Yeni telefonun bağlantısını kesin
  10. 10. Yeni telefonda HD SMTH'yi açın

При настройке нового телефона большинство людей используют инструмент переноса данных. Как перенести скрытые файлы из HD SMTH?

Если вы еще не использовали инструмент переноса телефона

Стандартные инструменты переноса обычно не переносят скрытые файлы. Чтобы ваши скрытые фото и видео были перенесены на новый телефон, выполните следующие действия:

  • 1. Перед использованием инструмента переноса телефона:
  • Откройте все файлы с помощью HD SMTH на старом телефоне
  • 2. Затем используйте инструмент переноса телефона
Причина: Инструменты переноса переносят только видимые фото и видео, а не скрытые данные.

Если вы уже использовали инструмент переноса телефона

Если вы уже использовали инструмент переноса, не восстановив сначала скрытые файлы, вы не найдете их на новом телефоне.

Есть два варианта их переноса.

Простой способ (только для телефонов)

  1. 1. На старом телефоне: Откройте HD SMTH → Откройте все файлы
  2. 2. На обоих телефонах: Установите и откройте приложение Local Send (Google Play Маркет)
  3. 3. На старом телефоне: Выберите открытые файлы в Local Send → Отправить
  4. 4. На новом телефоне: Примите файлы в Local Send
  5. 5. После переноса: Откройте HD SMTH на новом телефоне → Скрыть эти файлы

Быстрый способ (требуется ПК с Windows)

  1. 1. Подключите старый телефон к ПК через USB-кабель
  2. 2. Выберите режим "Передача файлов" на телефоне
  3. 3. На ПК: Откройте Проводник → Этот компьютер
  4. 4. Скопируйте папку .pg из корневого каталога телефона на ПК
  5. 5. Отключите старый телефон
  6. 6. Подключите новый телефон к ПК
  7. 7. Выберите режим "Передача файлов"
  8. 8. Вставьте папку .pg в корневой каталог нового телефона
  9. 9. Отключите новый телефон
  10. 10. Откройте HD SMTH на новом телефоне

새 휴대폰을 설정할 때 대부분의 사람들은 데이터 마이그레이션 도구를 사용합니다. HD SMTH의 숨겨진 파일은 어떻게 전송하나요?

휴대폰 마이그레이션 도구를 아직 사용하지 않은 경우

표준 전송 도구는 일반적으로 숨겨진 파일을 옮기지 않습니다. 숨겨진 사진과 동영상을 새 휴대폰으로 전송하려면 다음 단계를 따르세요:

  • 1. 휴대폰 마이그레이션 도구를 사용하기 전에:
  • 구형 휴대폰에서 HD SMTH를 사용하여 모든 파일을 숨김 해제하세요
  • 2. 그런 다음 휴대폰 마이그레이션 도구를 사용하세요
이유: 마이그레이션 도구는 보이는 사진과 동영상만 전송하며, 숨겨진 데이터는 전송하지 않습니다.

이미 휴대폰 마이그레이션 도구를 사용한 경우

이미 마이그레이션 도구를 사용했지만 숨겨진 파일을 먼저 복원하지 않은 경우, 새 휴대폰에서 숨겨진 파일을 찾을 수 없습니다.

이를 전송하는 두 가지 방법이 있습니다.

간단한 방법 (휴대폰 전용)

  1. 1. 구형 휴대폰에서: HD SMTH 열기 → 모든 파일 숨김 해제
  2. 2. 양쪽 휴대폰에: Local Send 앱 설치 및 실행 (Google Play 스토어)
  3. 3. 구형 휴대폰에서: Local Send에서 숨김 해제된 파일 선택 → 전송
  4. 4. 새 휴대폰에서: Local Send에서 파일 수락
  5. 5. 전송 후: 새 휴대폰에서 HD SMTH 열기 → 해당 파일 숨기기

빠른 방법 (Windows PC 필요)

  1. 1. 구형 휴대폰을 USB 케이블로 PC에 연결
  2. 2. 휴대폰에서 "파일 전송" 모드 선택
  3. 3. PC에서: 파일 탐색기 → 내 PC 열기
  4. 4. 휴대폰의 루트 디렉토리에서 .pg 폴더를 PC로 복사
  5. 5. 구형 휴대폰 연결 해제
  6. 6. 새 휴대폰을 PC에 연결
  7. 7. "파일 전송" 모드 선택
  8. 8. .pg 폴더를 새 휴대폰의 루트 디렉토리에 붙여넣기
  9. 9. 새 휴대폰 연결 해제
  10. 10. 새 휴대폰에서 HD SMTH 열기

新しいスマートフォンを設定する際、ほとんどの人はデータ移行ツールを使用します。HD SMTHの隠しファイルはどのように転送しますか？

スマートフォンの移行ツールをまだ使用していない場合

標準的な転送ツールは通常、隠しファイルを移行しません。隠し写真や動画を新しいスマートフォンに転送するには、次の手順に従ってください:

  • 1. スマートフォンの移行ツールを使用する前に:
  • 旧スマートフォンでHD SMTHを使用してすべてのファイルを表示に設定
  • 2. その後、スマートフォンの移行ツールを使用
理由: 移行ツールは表示可能な写真や動画のみを転送し、隠しデータは転送しません。

すでにスマートフォンの移行ツールを使用した場合

すでに移行ツールを使用し、隠しファイルを最初に復元しなかった場合、新しいスマートフォンで隠しファイルを見つけることができません。

それらを転送するには2つの方法があります。

簡単な方法（スマートフォンのみ）

  1. 1. 旧スマートフォンで: HD SMTHを開く → すべてのファイルを表示に設定
  2. 2. 両方のスマートフォンに: Local Sendアプリをインストールして開く（Google Playストア）
  3. 3. 旧スマートフォンで: Local Sendで表示に設定したファイルを選択 → 送信
  4. 4. 新しいスマートフォンで: Local Sendでファイルを受信
  5. 5. 転送後: 新しいスマートフォンでHD SMTHを開く → それらのファイルを非表示に設定

高速な方法（Windows PCが必要）

  1. 1. 旧スマートフォンをUSBケーブルでPCに接続
  2. 2. スマートフォンで"ファイル転送"モードを選択
  3. 3. PCで: ファイルエクスプローラー → このPCを開く
  4. 4. スマートフォンのルートディレクトリから.pgフォルダをPCにコピー
  5. 5. 旧スマートフォンを切断
  6. 6. 新しいスマートフォンをPCに接続
  7. 7. "ファイル転送"モードを選択
  8. 8. .pgフォルダを新しいスマートフォンのルートディレクトリに貼り付け
  9. 9. 新しいスマートフォンを切断
  10. 10. 新しいスマートフォンでHD SMTHを開く

Quando si configura un nuovo telefono, la maggior parte delle persone utilizza uno strumento di migrazione dei dati. Come si trasferiscono i file nascosti da HD SMTH?

Se Non Hai Ancora Utilizzato lo Strumento di Migrazione del Telefono

Gli strumenti di trasferimento standard in genere non migrano i file nascosti. Per assicurarti che le tue foto e video nascosti vengano trasferiti sul nuovo telefono, segui questi passaggi:

  • 1. Prima di utilizzare lo strumento di migrazione del telefono:
  • Rendi visibili tutti i file utilizzando HD SMTH sul tuo vecchio telefono
  • 2. Poi utilizza lo strumento di migrazione del telefono
Motivo: Gli strumenti di migrazione trasferiscono solo foto e video visibili, non i dati nascosti.

Se Hai Già Utilizzato lo Strumento di Migrazione del Telefono

Se hai già utilizzato lo strumento di migrazione senza prima ripristinare i file nascosti, non troverai i file nascosti sul nuovo telefono.

Ci sono due opzioni per trasferirli.

Metodo Semplice (Solo Telefoni)

  1. 1. Sul vecchio telefono: Apri HD SMTH → Rendi visibili tutti i file
  2. 2. Su entrambi i telefoni: Installa e apri l'app Local Send (Google Play Store)
  3. 3. Sul vecchio telefono: Seleziona i file resi visibili in Local Send → Invia
  4. 4. Sul nuovo telefono: Accetta i file in Local Send
